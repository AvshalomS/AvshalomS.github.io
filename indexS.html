Observe vacations
Observe vacations - React Nodejs & MySQL
About the project
  • Any user who logs in can view the vacations and tag them using the follow button.
  • Tagged vacation will change in real time for all users connected to the system - (Socket io).
  • The system administrator is responsible for the day-to-day management of the system, adding vacations, changing and deleting vacations.
  • The system administrator will also be able to display a real-time graph of the number of followers of the various vacations.
Technologies
  • HTML5 Structure, CSS3, Bootstrap design.
  • React, Redux thunk, middleware, Container components / service component / router etc.. .
  • NodeJS Using express, Restfull App, Socket io, MySql Design & create schema, Queries.

Cryptonite
Cryptonite - JQuery-AJAX RESTful API
About the project
  • The goal of the project is to access information from the world of virtual commerce (Bitcoin etc).
  • The home page contains information cards about virtual currencies (clicking "More info" will make a new call for information from the server, only if more than two minutes have passed since the last call).
  • Clicking the "Chart button" will add the currency to real-time reports, (up to 5 currencies can be selected).
  • The "Live Report screen" contains a graph of real-time selected currency rates.
Technologies
  • HTML5, CSS3, Bootstrap, Parallax scrolling, Collapser, progress bar.
  • Dynamic page layouts, Single Page Application foundations.
  • JavaScript, jQuery, Ajax (RESTful API), Callbacks, JSON.
